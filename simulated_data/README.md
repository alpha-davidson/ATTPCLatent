This directory is for the extraction of simulated data and plotting its events. 

Before running the extraction script, modify `run_data_extraction.sh` to specify the name of the file to be extracted. Additionally, create a `voxel_data` folder and insert the file there. 

Before plotting the events of extracted data, modify `run_plotting.sh` to specify the name of the file whose events will be plotted. A `plots` folder will be generated containing the plots of a specified file.
