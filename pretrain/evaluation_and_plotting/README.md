Use `evaluating_jigsaw.sh` to run the `evaluate_jigsaw_reconstruction.py` file, which will generate the histogram of reconstruction accuracy and sample reconstructed events and store them in the `{experiment}_plots` folder. `pointnet_model.py` and `plotting.py` are the main execution scripts, which are being called in the evaluation pipeline. `model_analysis.ipynb` notebook contains functionality for percent accuracy histogram analysis for further model evaluation.
