Modify the `unscrambling_jigsaw.sh` shell script to specify the beam, num-classes (number of classes, 24 voxel ids), etc. Submit the shell script to run the `pretrain_on_jigsaw.py` script. `pointnet_model.py` and `plotting.py` are the main execution scripts, which are being called in the training pipeline. The subsequent two folders will be generated after running the training pipeline:

- ```{experiment}_models```: The full model and the model weights generated by running the ```pretrain_on_jigsaw.py``` script. This folder is located inside the `training` folder, the same folder where the training pipeline is located.
- ```{experiment}_plots```: The learning curve. This folder is located inside the `evaluation_and_plotting` folder. The path from the current `training` to the plots is: `../evaluation_and_plotting/{experiment}_plots`. Later, after running the script for model evaluation, more plots will be generated and stored inside the same folder.
